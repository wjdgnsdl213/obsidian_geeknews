# AI 도입 ROI 사전진단 서비스

## 문제정의
소상공인·소규모 조직은 "AI 도입하면 좋다더라"는 막연한 정보만으로 ChatGPT/Copilot 구독을 시작했다가, [[우리 직장의 LLM 집단 망상]]에서 묘사된 것처럼 "전사 LLM 프로젝트 전부 실패", "1페이지 메뉴를 ChatGPT에 업로드해 묻기" 같은 무의미한 활용에 그치고 효과 없이 비용만 발생한다. 어떤 업무가 AI에 적합하고 어떤 업무는 오히려 손해인지 사전에 판단해주는 서비스가 없다.

## 해결방법
업무를 인터뷰/설문으로 수집한 뒤, [[생산적인 개인이 생산적인 기업을 만들지는 않는다]]의 "Institutional AI 7축"(조율/신호/객관성/우위/성과/실행지원/선제성) 프레임으로 각 업무를 점수화해, "AI 도입 적합" / "보류" / "오히려 손해" 3단계로 분류한 진단 리포트를 자동 생성한다. [[DemandProof – 만들기 전에 팔릴지 AI로 검증해주는 서비스]]의 "만들기 전에 AI로 검증" 컨셉을 "AI 도입 전에 AI로 검증"으로 변형한 메타적 포지셔닝.

## 사용데이터
- 업무 인터뷰/설문 응답 (정성 데이터)
- (선택) 기존 업무 처리 시간/오류율 로그
- Institutional AI 7축 평가 루브릭 (자체 정의)

## 핵심기능
- 업무 목록 입력 → 자율성 레벨(L1~L4, [[AI-네이티브 스타트업을 만드는 방법]] 참고) 자동 분류
- 7축 루브릭 기반 점수화 + "오류율 허용 가능 여부" 판단
- PDF/웹 형태의 진단 리포트 자동 생성 (적합 업무 / 부적합 업무 / 우선 자동화 후보 Top 3)

## 차별점
- 기존 AI 컨설팅은 "어떤 AI 툴을 쓸지"에 집중하지만, 이 서비스는 "AI를 쓸지 말지, 어디에 쓸지"를 먼저 판단 — [[우리 직장의 LLM 집단 망상]]이 보여준 실패 패턴을 정면으로 겨냥
- 진단 자체를 AI로 자동화해 저비용·즉시 제공 가능

## MVP범위
- 인턴 업무(상권분석 플랫폼)의 반복 업무 5~10개를 직접 7축으로 진단해보는 셀프 케이스 스터디
- Claude API로 "업무 설명 → 7축 점수 + 권고"를 자동 생성하는 간단한 웹폼/스크립트
