# DemandProof – 만들기 전에 팔릴지 AI로 검증해주는 서비스

**원문:** https://news.hada.io/topic?id=30426

## 요약
아이디어 입력 시 AI가 시장 수요를 Build/Pivot/Kill로 판정하는 서비스

## 핵심 포인트
- 아이디어나 URL을 입력하면 AI가 실시간 웹 데이터를 기반으로 60초 이내 Build/Pivot/Kill 판정, 수요 점수·근거, 타겟 페르소나, 첫 유저 채널 추천을 제공
- 가격 모델: Free(월 2회), Credit($4.99/10회), Day Pass($10/24시간), Pro($19/월) — Google OAuth 로그인만 지원
- 1인 개발자가 만들어 Product Hunt 론칭을 준비 중인 초기 단계 Show GN 프로젝트

## 왜 중요한가
"만들기 전에 AI로 시장성을 검증"하는 컨셉은 정훈의 "소상공인 AI 컨설팅" 사업 아이디어와 정확히 맞아떨어지는 구조(아이디어 입력 → 실시간 데이터 기반 수요 분석 → 채널 추천)이며, 가격/플랜 설계까지 참고할 수 있는 실제 운영 사례라 공모전·포트폴리오·사업 아이템 모두에 즉시 활용 가능하다.

## 내 관심 분야 적용
- 취업/면접: "AI 기반 시장 검증 서비스" 트렌드를 데이터 분석/AI 활용 사례로 언급
- 공모전: 유사 컨셉을 소상공인 창업 아이템(상권분석 데이터 + AI 수요 판정)으로 변형해 공모전 출품 아이디어로 발전
- 포트폴리오: 인턴 업무(상권분석 플랫폼)와 결합해 "지역 상권 데이터 기반 창업 아이템 Build/Pivot/Kill 판정 MVP"를 직접 구현
- 사업: 소상공인 대상 "내 아이템이 이 지역에서 팔릴까?"를 AI로 진단하는 서비스의 직접적 레퍼런스 모델
