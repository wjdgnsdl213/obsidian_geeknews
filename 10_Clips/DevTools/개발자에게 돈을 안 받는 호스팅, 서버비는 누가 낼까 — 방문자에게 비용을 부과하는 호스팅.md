# 개발자에게 돈을 안 받는 호스팅, 서버비는 누가 낼까 — 방문자에게 비용을 부과하는 호스팅

**원문:** https://news.hada.io/topic?id=30377

## 요약
개발자 무료, 방문자가 비용 부담하는 신개념 호스팅 플랫폼 coders.kr

## 핵심 포인트
- coders.kr은 Claude Code에 MCP로 연결("claude mcp add coders --transport http https://mcp.coders.kr/mcp")해 "2048 게임 만들어서 배포해줘" 한 마디로 즉시 배포 가능한 한국 Show GN 서비스
- 비용 구조가 역전: 개발자는 무료 배포, 방문자에게 사이트당 약간의 크레딧 제공(광고로 충전), 무거운 백엔드/LLM 호출은 방문자가 비용 부담, scale-to-0으로 유휴 프로젝트 비용 최적화
- llms.txt에 AI 에이전트용 지침을 넣어 Claude Code가 바로 이해하고 배포할 수 있도록 설계, 템플릿 저장소(template-coders) 제공, 향후 광고 추가 시 방문자 발생 즉시 수익화 예정

## 왜 중요한가
"AI 에이전트가 직접 배포까지 수행"하는 워크플로(MCP+llms.txt)는 [[Loop Engineering]]의 connector/automation 개념과 맞닿아 있고, "비용을 방문자에게 전가하는 프리미엄 호스팅" 모델은 소상공인을 위한 무료/저비용 웹사이트 제작 컨설팅 서비스를 설계할 때 참고할 수 있는 비용구조 아이디어다.

## 내 관심 분야 적용
- 취업/면접: "AI 에이전트가 MCP를 통해 배포까지 자동화하는 최신 개발 워크플로" 사례로 언급
- 공모전: 1인 개발자/학생 대상 "AI에게 말만 하면 배포되는" 무료 호스팅+크레딧 구조를 활용한 교육용 플랫폼 아이디어
- 포트폴리오: 이 GeekNews Vault에서 생성한 리포트나 미니 프로젝트를 coders.kr에 MCP로 배포해 "AI 에이전트 배포 자동화" 데모로 시연
- 사업: 소상공인 홈페이지를 "개발자(소상공인) 무료, 방문자 광고/크레딧 기반"으로 운영하는 저비용 웹사이트 제작 컨설팅 모델의 참고 사례

## 원문 스크랩

**Show GN: coders.kr** — Claude Code에 `claude mcp add coders --transport http https://mcp.coders.kr/mcp`로 MCP를 연결한 뒤 "2048게임 만들어서 coders.kr에 배포해줘"라고 입력하면 배포된 링크를 받을 수 있는 서비스.

**비용 구조**: 개발자는 무료로 사이트를 배포하고, 방문자가 사용하는 만큼의 비용을 충당하는 구조. 방문자에게는 사이트당 약간의 크레딧이 주어지며 광고를 통해 충전 가능 — 정적 호스팅과 단순 요청은 대부분 이 크레딧 안에서 해결됨. 무거운 백엔드나 LLM 호출이 있는 서비스는 방문자가 비용을 지불. scale-to-0 최적화로 쉬고 있는 프로젝트의 비용을 절감.

**사용법**: llms.txt에 AI 에이전트용 지침을 포함시켜 Claude Code가 바로 이해하도록 설계. MCP 연결과 coders.kr 가입 필요. 템플릿 저장소(github.com/cykim8811/template-coders) 제공. 개발 예정 기능으로 광고를 추가하면 방문자 발생 시 즉시 수익화 가능. 작성자는 막 배포한 서비스이며 피드백을 요청.
